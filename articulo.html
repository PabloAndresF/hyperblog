Git es un sistema libre y de código abierto distribuido de control de versiones diseñado para manejar todo,
desde pequeñas a proyectos muy grandes con rapidez y eficacia.
Git es fácil de aprender y tiene una pequeña huella con un rendimiento increíblemente rápido.
Se supera a herramientas de SCM como Subversion, CVS, Perforce y ClearCase con características como barato
ramificación locales, áreas de almacenamiento convenientes, y múltiples flujos de trabajo.
